Crypto Note PWA

How to use:
1. Upload all files to any HTTPS web host (or run locally on localhost).
2. Open index.html in Chrome on Android or Safari on iPhone.
3. Android: use the install button when it appears.
4. iPhone: Share -> Add to Home Screen.

Important:
- For installability on Android, the app must be served over HTTPS or localhost.
- iPhone Home Screen web apps are added from Safari.
